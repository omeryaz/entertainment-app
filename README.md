# Entertainment Web App

## About

A streaming platform clone where you can navigate between different pages (Home, Movies, TV Series, and Bookmarked Shows), add/remove bookmarks, search for shows.

<br>

## Built With

    - React
    - TypeScript
    - React Router
    - Tailwind CSS
    - Vite
    - Framer Motion
    - Responsive design
